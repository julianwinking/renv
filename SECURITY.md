# Security policy

## Supported versions

renv is pre-1.0. Only the latest commit on `main` receives security fixes.

## Reporting a vulnerability

Please do not open public issues for security problems.

Report vulnerabilities privately via
[GitHub security advisories](https://github.com/julianwinking/renv/security/advisories/new)
or by email to julian.winking@gmail.com. Include a reproduction if you can.

You will get an acknowledgement within a few days. Please allow a reasonable
window for a fix before public disclosure.
