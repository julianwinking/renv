# {{title}}

A research project in the `renv` auto-research environment. Created {{date}}.

This folder is its **own git repository** (link your own GitHub remote) and is
gitignored by the parent research-env repo. The shared paper corpus, the central
research database, and the `renv` engine live in the parent environment; this repo
holds only *this paper*: its plan, code, manuscript, and run artifacts.

## Layout
- The research *plan* lives in the store, not here: thesis/contributions are
  claims, risks are questions, evaluation design is a decision entry — see the
  cockpit graph or `renv claim list {{slug}}` / `renv log list {{slug}}`.
- `src/` — experiment code (entrypoints honoring the `RENV_RUN_DIR`/`RENV_PARAMS`/`metrics.json` contract).
- `text/` — the manuscript (`paper.tex`); `results_table.tex` + `references.bib` are GENERATED by `renv weave`.
- `runs/` — per-run artifacts (figures, logs); large/derived, not the source of truth.
- `reviews/` — saved `renv review` reports.

## Workflow
```bash
renv exp new {{slug}} 001-baseline --hypothesis "..."   # add an experiment (DAG node)
renv exp run {{slug}} 001-baseline --entrypoint src/run.py --param k=8
renv cite "<claim>" {{slug}} --write                     # span-anchored, verified citation
renv weave {{slug}}                                      # regenerate numbers + bib from the store
renv review {{slug}}                                     # automated per-section checks
```
See the parent `AGENTS.md` for the operating protocol.
